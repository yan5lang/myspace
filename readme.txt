网络时代的浮华

模仿
这个简单的技巧正是这篇文章的核心，模仿就是在参与对话的过程中拷贝其他人的心理上的惯用风格、节奏和面部表情。你变成了其他人的镜子中的形象。

模仿在社会交往中很自然的存在着，只有当你意识到它的存在并且认识到了它的作用的时候，它才能被当作一个有效沟通的工具和一个产生和睦关系的工具而被加以应用。

在某种层次上说，惟妙惟肖的模仿别人可以使你感觉到他们的感受。我曾经做过一个试验，在一个三个人一组的研讨会上，第一个人以描述一个场景开始，他把他看到的、感受到的以及体验到的表述出来。第二个人单纯模仿这个人的面部表情和肢体动作，第三个人则用他自己调整过的面部表情和肢体语言来表达和第一个人一样的感受，几分钟后，第二个人解释了他感受到了什么，他不仅仅感觉到了第一个人的感觉，并且有些部分还感受到了第一个人的想象。我自己在做完这个实验后十分高兴。

下一次，你和某人参于到对话中的时候，尝试着模仿肢体语言，手势和面部表情，你将会发现谈话立刻就变得非常友好和坦诚。

举个例子来说，你和某些人隔着桌子面对面坐着。你看见他们用左手轻轻的倾身向前拿起一杯水放到右边，你模仿他们这样倾身向前用右手把水拿到左边。

下次试试，就当是好玩的。:)
